Changelog

v3.9.15

Fixes:

  • Fix publicacao duplicada: dashboard rodava scheduler embutido em paralelo com scheduler.py
  • Fix yt-dlp: fallback sem cookies quando Firefox retorna "page needs to be reloaded"
  • Fix re-publicacao: clips com erro_upload/publicando nao sao re-tentados automaticamente

v3.9.14

Features:

  • LLM chain: 3 providers em cadeia para gerar prompt de thumbnail (fallback automatico)
  • Multi-provider de imagem: Piramyd + Kie.ai (GPT-4o, Flux Kontext)
  • Kie.ai API key no painel + campo modelo de imagem como texto livre
  • Config append: novas chaves sao adicionadas automaticamente na planilha

v3.9.13

  • Painel clips: coluna Link mostra URL real da publicacao no YouTube
  • Botao excluir ao lado da privacy (sempre visivel)
  • Icone arquivo mudou de pasta para play (▶)
  • Excluir clips pendentes (remove do manifest)

v3.9.12

Features:

  • Lock na planilha: marca "publicando" antes de enviar, evita duplicatas entre processos
  • Health bar - status do sistema (yt-dlp, Sheets, APIs, YouTube, Scheduler)
  • Proximos horarios de corte/pub visiveis no dashboard
  • Toggle "Link da live na descricao" nas configuracoes
  • Corte roda em thread separada - nao bloqueia publicacao
  • Changelog acessivel pelo numero da versao
  • Sync rapido com 1 clique (sem filtros) + sync avancado separado
  • Auto-sync: toggle ON/OFF, sincroniza lives faltantes a meia-noite

Bugs corrigidos:

  • Fix contador clips_publicados contando em dobro
  • Fix crash no startup (UnboundLocalError config)
  • Fix flood de log do yt-dlp travando terminal (filtro a cada 10%)
  • Fix link da live na descricao retornando {video_id} literal
  • Fix minuto do horario nao carregando no dashboard
  • Fix publicacao perdida quando corte demorava
  • Services systemd apontando para diretorio correto
